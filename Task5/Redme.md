## Task5 – Laundry Service
## Project Description

This project is a simple landing page for a laundry service created using HTML and CSS .
The goal of the project was to practice building a webpage layout using semantic HTML structure and CSS positioning without using Flexbox.

The webpage includes a navigation bar, a hero section with promotional text, and an illustration image representing laundry services. The page also includes hover effects to improve user interaction.


## Requirements Met

The following requirements were implemented in this project:

* Created the HTML structure manually using proper element nesting.
* Built a navigation bar with:

  * Logo on the left
  * Menu links centered
  * User profile button on the right
* Implemented a two-column hero section layout:

  * Left side contains heading, description, and button
  * Right side contains a laundry illustration image

* Applied proper spacing, padding, and alignment for a clean layout.
