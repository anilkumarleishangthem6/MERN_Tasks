This project is a simple and visually clean login page created using HTML and CSS.
The layout is designed using Flexbox to create a split-screen interface with an image on the left and a login form on the right.

The page is centered vertically and horizontally on the screen and follows a modern UI design with a clean structure.

##Features##

Split layout design 

Page centered using Flexbox

Full screen height layout using 100vh

Styled input fields and login button

Hover effect for the login button and links

Clean and organized UI
