# Responsive Laundry - Task 7

## Project Description

This project is a responsive version of the Laundry Service created in the previous task, Task5. The page layout was improved using CSS Media Queries so that it works properly on desktop, tablet, and mobile devices without breaking the design.

## Features Implemented

## Navbar

* Logo, menu links, and username button aligned properly.
* Tablet View: Text sizes of menu links, logo, and username button are reduced.
* Mobile View: Navigation links are hidden, only the logo and username button are displayed.

## Hero Section

* Hero section contain-text content on the left and image on the right.
* Tablet Vie Text sizes and image size are reduced to fit the screen.
* Mobile View: Layout changes to column format, placing the text section on top and image section at the bottom.
* Button and spacing are adjusted for better readability on smaller screens.
