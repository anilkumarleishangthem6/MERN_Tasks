# Thala 7 Sneaker Store - CSS Mini Project (task11)

An e-commerce web application layout built entirely from scratch. This project demonstrates modern UI design principles by featuring a seamless scrolling experience through a complete user journey: from a landing page down to product details, shopping cart, and a checkout form.

Built strictly with HTML and CSS, this project relies heavily on CSS Flexbox to achieve complex, pixel-perfect alignments without the use of Bootstrap or any heavy external styling frameworks.

# Features
Single Page Application (SPA) Layout: Smooth vertical stacking of all main e-commerce sections.

Dynamic Hero Section: Eye-catching landing area with an angled product showcase.

Structured Shopping Cart: Clean tabular layout for cart items paired with a sticky-style order summary card.

Compact Checkout Form: Highly optimized, one-screen checkout form with integrated payment tabs and trust badges.

Fully Custom UI: Styled entirely with custom CSS.

# Tools
