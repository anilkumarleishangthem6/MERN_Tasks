# JavaScript Task15

This project demonstrates basic operations on an array using different types of functions in JavaScript.

## Array Used
```js
let arr = [8, 9, 5, 2, 7, 10];
```

### Functions Used
## 1. Find Maximum Number
Uses Math.max() with spread operator
Returns the largest number in the array

## 2. Find Sum of Elements
Uses a for loop
Adds all elements of the array

## 3. Count Odd Numbers
Uses a loop and condition
Counts how many odd numbers are present
