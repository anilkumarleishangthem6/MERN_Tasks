# JavaScript Loop-Based Programs

This project contains basic JavaScript programs implemented using loops.

---

##  Features

### 1. Sum of First N Numbers
Calculates the sum from 1 to `n`.

### 2. Multiplication Table
Prints the table of a given number up to 10.

### 3. Prime Number Check
Checks whether a number is prime or not.

### 4. Factors of a Number
Displays all factors of a given number.

### 5. Sum of Digits
Finds the sum of digits of a number.

### 6. Armstrong Number Check
Checks whether a number is an Armstrong number.
## used math.floor to calculate the floor dividion to check the Armstrong number.
