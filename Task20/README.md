# Task20 (Laundry Service Web Application)

A modern, responsive, single-page front-end web application for a laundry service. This project features a dynamic booking system, a clean user interface, and smooth interactive elements built entirely with JavaScript, HTML5, and CSS3.

##  Features

* **Dynamic Cart System:** Users can add and remove laundry services from their cart. The cart automatically updates the total price and calculates the item count.
* **Empty State Handling:** Displays a clean, visually appealing "Empty Cart" message when no services are selected.
* **Booking Form Simulation:** Includes a user details form that validates input and displays a dynamic success message upon booking, followed by an automatic cart reset.
* **Smooth Scrolling Navbar:** Sticky navigation bar with a modern center-out hover underline effect and smooth anchor scrolling to different page sections.
* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices using CSS media queries.
* **Modern UI Components:** Features a hero section, achievements banner, quality description features, a newsletter subscription box, and a comprehensive footer with social icons.

## Technologies Used

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom styling, Flexbox layouts, gradients, and CSS transitions/animations.
* **Vanilla JavaScript (ES6):** DOM manipulation, state management for the cart, and event handling.
* **Ionicons:** Scalable vector icons for the footer social links.
