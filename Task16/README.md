# Task16 Student Records Web App

## Objective

A simple web page to display and search student records using **HTML, CSS, and JavaScript**.

---

## Features

* Display student details in **card format**
* Dynamic rendering using **JavaScript (map)**
* Real-time search using **filter**
* Responsive and clean UI

---

## Technologies Used

* HTML
* CSS
* JavaScript

---

## 📊 Student Data

Each student contains:

* Name
* Marks
* Class
* Address

---

## Functionality

* Student cards are generated dynamically using `map()`
* Search input filters students by name using `filter()`
* Results update instantly as user types


##  Output

Displays student records in card layout with a working search bar.
